How it works

Images are processed locally in your browser using the **Canvas API**. Quality and format choices affect size. No files leave your device.

Tips:

  • Use **WebP** for best compression if supported by target devices.
  • Lower quality to reduce file size (**0.6 - 0.8 is a good start**).
  • Resize to max width to significantly reduce bytes for large photos.
